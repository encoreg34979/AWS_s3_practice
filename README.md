# AWS CLI

aws configure list 查看當前使用者資料

aws s3 ls 列出 s3 的bucket

aws s3 cp --recursive s3://source資料夾 s3://target資料夾    
--recursive 會掃描整個資料夾的檔案 與 cp 一起用的話就是會全部複製過去
